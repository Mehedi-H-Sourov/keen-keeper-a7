# 🤝 KeenKeeper

## 📌 Project Description
KeenKeeper is a relationship management web app that helps you stay connected with your friends by tracking interactions like calls, texts, and video chats. It allows users to monitor their communication habits, view timelines of interactions, and analyze engagement through visual statistics.

---

## 🚀 Technologies Used
- Next.js (App Router)
- React
- Tailwind CSS
- Recharts (for data visualization)
- React Context API (state management)
- LocalStorage (data persistence)

---

## ✨ Key Features

### 1. 📊 Interaction Tracking System
Users can log interactions (Call, Text, Video) directly from the friend details page, which are stored and reflected across the app.

### 2. 🕒 Dynamic Timeline
A timeline page displays all interactions in real-time with filtering options, allowing users to easily review their communication history.

### 3. 📈 Analytics Dashboard
The stats page visualizes interaction data using a pie chart, giving users insights into their communication patterns.
